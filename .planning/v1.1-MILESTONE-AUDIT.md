## Critical Gaps (Blockers)

### GAP-1 (CRITICAL): verify-promotion missing `build` in `needs` — TAG always empty

**File:** `scripts/generate-workflow.sh` line 260
**Symptom:** `verify-promotion` job depends on `[deploy-staging, deploy-production]` but not on `build`. GitHub Actions only populates `needs.<job>.outputs.*` for jobs explicitly in the current job's `needs` array. `TAG: ${{ needs.build.outputs.tag }}` resolves to empty string.

**Impact:** The assertion `if [ "$actual" = "$TAG" ]` always fails (actual SHA ≠ ""), so `verify-promotion` always exits 1 on any successful real deploy. `ghcr-cleanup` — which depends on `verify-promotion` — never runs. GHCR tag retention is permanently disabled for any repo using this generated workflow.

**Requirements affected:** PROMOTE-01 (assertion broken), PROMOTE-02 (ghcr-cleanup blocked by false failure, not real divergence), INV-04 (documented enforcement does not function)

**Fix:** `scripts/generate-workflow.sh` line 260:
```
needs: [deploy-staging, deploy-production, build]
```

---

### GAP-2 (HIGH): Contract test C9 blanket-rejects all build-args including identity-only ones

**File:** `test/validate-workflow-contract.sh` line 169
**Symptom:** C9 check: `if grep -qE '^\s*build-args:' "$WF"; then fail "C9..."` — rejects any `build-args:` block. Phase 07 (LAYER3-01) added `build-args: | GIT_SHA=... BUILD_TIMESTAMP=...` to the generated workflow. The contract test now always fails C9 for any workflow generated after Phase 07.

**Impact:** `bash test/validate-workflow-contract.sh` exits non-zero on every run. CI self-check cannot pass. LAYER3-01 implementation is correct; the test's policy is too broad.

**Requirements affected:** LAYER3-01 (implementation correct; contract self-verification broken)

**Fix:** Refine C9 to allow GIT_SHA and BUILD_TIMESTAMP while still rejecting env-specific values:
```bash
if grep -qE '^\s*build-args:' "$WF"; then
  # Identity-only build-args (GIT_SHA, BUILD_TIMESTAMP) are allowed
  env_specific=$(grep -A5 'build-args:' "$WF" | grep -v 'GIT_SHA\|BUILD_TIMESTAMP' | grep -c '=') || true
  if [ "$env_specific" -gt 0 ]; then
    fail "C9: env-specific build-args present — breaks same-image promotion"
  fi
fi
```
