Extend init.sh to prompt operators for an optional deploy_server value (default
blank = unchanged behavior) and extend the coolify.yaml template to render the
field as a commented placeholder when blank or as an active assignment when set.
This satisfies MSRV-01 from the user-facing init flow; the runtime
implementation lives in plan 04-01.

feature without any manual yaml editing. The default (Enter to skip) preserves
the current "deploy to localhost" experience.

substitution, and init/templates/coolify.yaml.tmpl with one new field block.
